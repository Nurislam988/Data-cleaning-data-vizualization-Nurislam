README: Metropolitan Museum of Art Open Access Subset
Overview
This dataset is a curated collection of 5,000 object records from The Metropolitan Museum of Art’s (The Met) Open Access database. It provides a diverse cross-section of the museum's holdings, ranging from ancient Egyptian artifacts and classical prints to modern photography.

Data Source
The data is derived from the official Metropolitan Museum of Art Open Access repository. It includes objects that are part of the public domain as well as selected copyrighted works, representing various departments and historical eras.

Dataset Structure
The file is structured to provide comprehensive information about each artwork, divided into the following categories:

1. Object Identification
Object Number and ID: Unique identifiers used by the museum.

Department: The specific museum department (e.g., Egyptian Art, Drawings and Prints, Photographs).

Title and Object Name: The name of the work and its general category (e.g., "Scarab," "Book," "Mugshot").

2. Artist and Creator Information
Artist Name and Role: Details about the creator, including their role (Author, Illustrator, Photographer).

Biographical Data: Artist nationality, birth and death dates, and gender where available.

External Links: Reference URLs to the Getty Union List of Artist Names (ULAN) and Wikidata for deeper biographical research.

3. Historical Context
Period and Dynasty: Information regarding the era of creation (e.g., "Second Intermediate Period").

Culture and Geography: The cultural origin and geographic location (Country, City, Region) where the object was created or found.

Object Date: The specific year or time range when the piece was produced.

4. Physical Description
Medium: The materials and techniques used (e.g., "Albumen silver print," "Glass negative," "Etching").

Dimensions: The physical size of the object, provided in both centimeters and inches.

5. Museum Metadata and Access
Public Domain Status: Indicates whether the work is in the public domain.

Highlights and Timeline: Flags indicating if the object is considered a "Museum Highlight" or part of the "Heilbrunn Timeline of Art History."

Tags: Subject matter keywords (e.g., "Men," "Portraits," "Flowers") with links to Art & Architecture Thesaurus (AAT) and Wikidata.

Link Resource: A direct URL to the object's official page on the Met Museum website.

Usage Notes
This CSV file is intended for educational purposes, data visualization, and art history research. It allows for the analysis of museum collection trends, such as the distribution of materials across centuries or the representation of different cultures within specific departments.

Note: This documentation is based on the metadata provided within the met_sample_5k.csv file.
