The Metropolitan Museum of Art - Open Access Data Sample (10k)
This repository contains a sample of 10,000 objects from The Metropolitan Museum of Art's Open Access collection. The data includes detailed metadata for a wide variety of artworks, ranging from ancient artifacts to modern prints.

📊 Dataset Overview
File Name: met_sample_10k.csv

Total Records: 10,000

Total Columns: 54

Temporal Range: ~11,000 BCE to 2025 CE

Data Source: The Metropolitan Museum of Art Open Access

🔍 Key Features
This dataset includes several high-value columns for analysis and visualization:

Identification: Object ID, Object Number, Title, Object Name

Artist Info: Artist Display Name, Artist Nationality, Artist Role, Artist Wikidata URL

Temporal Data: Object Begin Date, Object End Date, AccessionYear

Categorization: Department, Culture, Medium, Classification

Public Domain Status: Is Public Domain (Boolean indicating if the work is in the public domain)

Resources: Link Resource (Direct link to the object on the Met's website)
